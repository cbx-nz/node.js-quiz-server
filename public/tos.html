4.5 Ban System and Enforcement

For serious violations, administrators and hosts may issue bans:

IP-Based Bans (Network-Level)

  • IP Bans: Block access from specific IP addresses
  • Ban Duration: Bans can be temporary (customizable hours) or permanent
  • Auto-Expiration: Temporary bans automatically expire and are removed by the server
  • Network Effect: In schools/organizations, bans affect entire network (all users on same public IP)
  • Ban Page: IP-banned users redirected to ip-banned.html showing ban details

UUID-Based Bans (Account-Specific)

  • Account Tracking: Each player assigned unique identifier stored in browser
  • Persistent Bans: UUID bans follow account across sessions and name changes
  • Host Requests: Game hosts can request UUID/IP bans for problematic players during gameplay
  • Admin Approval Required: All ban requests must be reviewed and approved by administrator
  • Ban Options: Admin can choose to ban UUID only, IP only, or both
  • Temporary Support: UUID bans can be set to expire after specified hours
  • Individual Targeting: UUID-only bans affect specific account, not entire network
  • Ban Page: UUID-banned users redirected to game-banned.html showing account ban details
  • Real-Time Enforcement: Approved bans immediately broadcast to all connected clients

Ban Enforcement

  • Immediate Redirect: Banned users automatically redirected to appropriate ban page
  • Page Blocking: Banned accounts cannot access host.html, presenter.html, creator.html, or docs.html
  • Restricted Access: While banned, you can ONLY view:
    • game-banned.html or ip-banned.html (your ban information with details)
    • tos.html (Terms of Service)
    • privacy.html (Privacy Policy)
  • Blocked Features: All game features, navigation modal, and protected pages are inaccessible
  • Account Tracking: Ban status stored and checked across all pages
  • Reasons for Bans: Harassment, prohibited conduct, ToS violations, malicious behavior, cheating
  • Appeal Process: Contact server administrator to appeal ban decisions
  • Circumvention: Attempting to bypass bans violates ToS and may result in additional restrictions
